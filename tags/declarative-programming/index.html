Out of the tar pit

“Tar pit” is an idiom used since 1970s based on the similarity between software complexity and tar pits trapping engineers and animals (resp.). Out of the tar pit is a paper that analyzes the causes of this complexity and classifies it in several categories. The authors also analyzes the approaches that each type of programming technique uses to sove this complexity problem. But the most important contribution the authors do is giving a solution in the form of a new paradigm of computer programming.
